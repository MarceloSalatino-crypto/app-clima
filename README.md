# 🌤️ App de Clima

Aplicativo de previsão do tempo que busca dados em tempo real.

## 🚀 Como usar
Acesse: [clique aqui](https://marcelosalatino-crypto.github.io/app-clima)

## 🛠️ Tecnologias
- HTML5, CSS3, JavaScript
- API OpenWeatherMap
- Fetch API + Async/Await

## ✨ Funcionalidades
- Busca clima por cidade
- Temperatura, umidade e vento
- Data e dia da semana atual
- Glassmorphism design
